行前注意事項

★

一定要先裝的 App

釜山自由行最實用工具

  • Naver Map：韓國導航最準
  • Naver Translate / Papago：翻譯最實用
  • Kakao T / Uber：叫車
  • WOWPASS：交通 / 付款 / 換匯
  • Blue Line Park：膠囊列車先預約
1

行李與使用提醒

出發前先確認

  • 護照與信用卡帶好，退稅要用
  • 行動電源隨身帶，不要託運
  • 韓國轉接頭記得準備
  • Spa Land 排在醫美之前
  • 醫美日行程排鬆一點，下午不要太累
2

吃東西與走路節奏

把路線排在同區域比較順

  • 海雲台那天可以把 Momos / miffy store / Haeundae 31 cm 一次解決
  • 南浦那天建議 醫美 + 札嘎其 + BIFF 一起排
  • 西面那天就專心 Wannabe + 選物店 + 咖啡店
